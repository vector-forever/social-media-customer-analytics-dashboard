# 📊 Social Media & Customer Analytics Dashboard

## 📖 Project Overview

This project is an interactive Power BI dashboard developed to analyze social media performance, customer conversion rates, and customer review sentiment across multiple products and time periods.

The dashboard helps businesses understand customer engagement, monitor product performance, track conversion efficiency, and evaluate customer satisfaction through advanced visual analytics.

The project combines data visualization, KPI tracking, sentiment analysis, and business intelligence techniques to provide actionable insights for decision-making.

---

# 🚀 Features

- Interactive Power BI dashboard
- Dynamic year and month filters
- Product-wise analysis
- Conversion funnel analysis
- Social media engagement tracking
- Customer review sentiment analysis
- KPI monitoring
- Heatmaps and trend analysis
- Drill-down and filtering functionality
- Multi-page dashboard navigation

---

# 📂 Dashboard Pages

## 1️⃣ Overview Dashboard

The Overview Dashboard provides a high-level summary of overall business performance.

### Includes:
- Overall Conversion Rate
- Total Views
- Total Clicks
- Total Likes
- Average Customer Rating
- Monthly Conversion Trends
- Product-wise Conversion Analysis
- Social Media Performance Trends
- Customer Rating Trends

---

## 2️⃣ Conversion Details Dashboard

The Conversion Details Dashboard focuses on customer journey and product conversion analysis.

### Includes:
- Conversion Funnel Visualization
- Product-wise Conversion Rate
- Monthly Conversion Trend
- Customer Journey Analysis
- Product Conversion Heatmap
- Purchase Funnel Metrics
- Conversion Matrix Analysis

---

## 3️⃣ Social Media Details Dashboard

The Social Media Dashboard analyzes engagement metrics and content performance.

### Includes:
- Views, Clicks and Likes Analysis
- Monthly Engagement Trends
- Product-wise Engagement Tracking
- Content Type Comparison
- Social Media Heatmap
- Performance Matrix

---

## 4️⃣ Customer Review Details Dashboard

The Customer Review Dashboard focuses on customer feedback and sentiment analysis.

### Includes:
- Average Rating Analysis
- Rating Distribution
- Sentiment Category Analysis
- Customer Review Table
- Sentiment Trends
- Review Scatter Plot
- Positive, Negative and Neutral Review Analysis
